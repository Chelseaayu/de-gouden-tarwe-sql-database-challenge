# 🌾 De Gouden Tarwe — SQL Database Challenge

![Status](https://img.shields.io/badge/Status-Complete-green)

> A multi-day SQL database challenge built around the fictional bakery **De Gouden Tarwe** ("The Golden Wheat"), designed to develop practical skills in relational database design, querying, and data analysis.

---

## 📖 About This Challenge

**De Gouden Tarwe** is a structured SQL learning challenge that simulates a real-world bakery business scenario. Through a series of progressively complex exercises, you will design databases, write queries, and extract meaningful insights from business data — all within the context of managing a bakery operation.

This challenge is ideal for students and professionals looking to strengthen their SQL and relational database skills through hands-on, scenario-based learning.

## 🚀 Getting Started

1. **Clone this repository**
   ```bash
   git clone https://github.com/Chelseaayu/de-gouden-tarwe-sql-database-challenge.git
   cd de-gouden-tarwe-sql-database-challenge
   ```

2. **Open the PDF files in order**, starting from Day 1 and working your way through each day sequentially.

3. **Set up your SQL environment** — you can use any of the following:
   - [PostgreSQL](https://www.postgresql.org/)
   - [MySQL](https://www.mysql.com/)
   - [DBeaver](https://dbeaver.io/) (GUI client)
   - [DB Fiddle](https://www.db-fiddle.com/) (browser-based)

4. **Follow the instructions** in each day's PDF to complete the exercises.

---

## 💡 What You Will Learn

- Designing normalized relational database schemas
- Writing Entity-Relationship Diagrams (ERDs)
- Creating and managing tables with appropriate data types and constraints
- Performing `SELECT`, `INSERT`, `UPDATE`, and `DELETE` operations
- Using aggregate functions (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
- Writing `JOIN` queries to combine data across multiple tables
- Building subqueries and complex query logic
- Generating business insights through data analysis
